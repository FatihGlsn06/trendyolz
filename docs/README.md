# n8n Academy - Interactive Learning Platform

> ADHD-friendly, Codecademy-style interactive n8n learning platform

## Features

### Learning System
- **Micro-lessons**: 3-5 minute focused lessons
- **Visual Roadmap**: Interactive skill tree showing progress
- **Instant Feedback**: Quiz results and XP rewards
- **Chunked Content**: Information broken into digestible pieces

### ADHD-Friendly Design
- **Pomodoro Timer**: Built-in focus mode with 25-minute sessions
- **Gamification**: XP system, levels, and badges for motivation
- **Progress Tracking**: Visual progress indicators everywhere
- **Break Reminders**: Automatic break notifications

### Interactive Features
- **Live Sandbox**: Build and test workflows without leaving the platform
- **Quick Challenges**: Bite-sized tasks to practice skills
- **Achievement System**: Unlock badges and track milestones

## Tech Stack

- Vanilla JavaScript (No framework dependencies)
- Tailwind CSS (CDN)
- LocalStorage for progress persistence
- Canvas API for confetti effects

## Getting Started

```bash
# Clone the repository
git clone <repo-url>

# Navigate to project
cd n8n-academy

# Start local server
npx serve .
```

## Deployment

### Vercel
